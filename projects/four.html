4Art

4Art is an NFT marketplace and the product has millions of users globally. It provides multiples services like NFT creation, buying and selling of NFT's. It has multiple microservices. I've worked on 4Art as a DevOps Engineer developing and managing docker images of the microservices using Docker/DockerHub. Wrote Terraform scripts for the repetitive configuration of the cloud resources like ec2,auto-scaling,ebs-volumes and S3 buckets.Monitoring the performance of the application using AWS cloudwatch. LAMP/MERN stack configuration on the cloud for the microservices using docker repositories.
