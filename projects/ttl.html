Type To Learn

Type to Learn is an educational typing program designed to enhance keyboarding skills, particularly for students. The program offers a structured curriculum with lessons, exercises, and engaging games to improve typing speed and accuracy through regular practice. It is widely used in educational settings to help individuals develop proficient touch typing abilities.

In this project, I took charge of orchestrating software deployments through AWS CodePipeline, ensuring a streamlined and automated delivery process. I actively maintained the Docker image of the application, utilizing proper tagging on DockerHub for effective version control. Additionally, I provided valuable support to developers by establishing Git and Bitbucket guidelines for efficient branching and tagging workflows.

Technology Stack

  • AWS Codepipeline
  • Docker
  • DockerHub
  • Git
  • Github/Bitbucket
